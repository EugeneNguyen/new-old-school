# REQ-002: REST API Core

## Description
A robust REST API backend that allows the frontend to interact with the local filesystem and system processes securely.

## Acceptance Criteria
- [x] Basic `/api/system` endpoint for health checks.
- [ ] Standardized JSON error response format.
- [ ] Middleware for basic request logging.
- [ ] Ability to execute local shell commands via API (with safety guards).

## Technical Constraints
- Must use Next.js Route Handlers.
- Response times should be < 200ms for local calls.
