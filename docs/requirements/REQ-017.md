# REQ-017: Copy Resume Command from Session

## Description
Add a "Copy command" affordance to each session in the Claude Terminal so users can grab a ready-to-paste `claude --resume <sessionId>` string and continue the conversation in an external terminal.

## Acceptance Criteria
- [x] Each session row in the Session Panel exposes a copy-command icon button.
- [x] Clicking it copies `claude --resume <sessionId>` to the clipboard without triggering session selection.
- [x] The active session also has a copy-command control near the terminal header.
- [x] Successful copy shows short inline visual feedback (icon swap to check, ~1.5s).
- [x] Uses `navigator.clipboard.writeText` with a safe fallback when unavailable.

## Technical Constraints
- Next.js App Router + TypeScript with `@/*` path aliases.
- shadcn/ui + Tailwind; Lucide icons (`Copy`, `Check`).
- No backend change; session ID already present in `SessionSummary`.
- Row-level button must stop event propagation so the row click still handles selection.
