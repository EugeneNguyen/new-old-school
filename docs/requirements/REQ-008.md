# REQ-008: Collapsible Sidebar Toggle

## Description
Add a toggle mechanism to the dashboard sidebar so users can collapse it to an icon-only view, freeing horizontal space for the main content area.

## Acceptance Criteria
- [x] A toggle button is visible on the sidebar to collapse/expand it
- [x] When collapsed, the sidebar shrinks to icon-only width (~64px) and hides text labels
- [x] When expanded, it returns to full 256px width with labels
- [x] The transition is smooth (using existing `duration-300`)
- [x] The main content area automatically fills the freed space
- [x] Collapsed/expanded state persists visually during navigation within the dashboard

## Technical Constraints
- Next.js App Router with React context for sidebar state
- Tailwind CSS for responsive width classes
- Lucide React icons for the toggle button
- shadcn/ui-style component conventions
