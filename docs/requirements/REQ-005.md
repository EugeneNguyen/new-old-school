# REQ-005: Claude Code Chat Terminal

## Description
Transform the existing Shell Terminal screen into a chat interface for Claude Code. The terminal input sends prompts to the `claude` CLI using the `-p` flag (non-interactive mode), captures output to session files under `.claude/sessions/`, and displays Claude's responses in the terminal UI.

## Acceptance Criteria
- [x] Backend API route (`/api/claude`) that spawns `claude -p "<prompt>"` and streams output via SSE
- [x] Session management u2014 each conversation tracked by `session_id` with `--resume` support
- [x] Terminal UI repurposed as a Claude chat interface with chat-style message display
- [x] Streaming mechanism to show Claude's response in realtime as it generates
- [x] Conversation history visible in the terminal scroll area
- [x] Loading/thinking state while Claude is generating a response

## Technical Constraints
- `claude` CLI must be available on the system PATH
- Uses `-p` flag for non-interactive piped mode with `--output-format stream-json --verbose`
- Next.js App Router, TypeScript, shadcn/ui + Tailwind
- Streaming via Server-Sent Events (SSE) pattern
- Session continuity via `--resume <session_id>` flag
