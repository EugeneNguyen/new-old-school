# REQ-007: Slash Skill Auto-Complete

## Description
Add a slash command system with auto-complete suggestions to the Claude chat terminal. When the user types `/` in the input field, a popup appears listing available slash skills, filtering as they type. Skills are defined in a registry file (`config/skills.json`).

## Acceptance Criteria
- [x] Slash skill registry exists at `config/skills.json` with name, description, and usage for each skill
- [x] Typing `/` in the terminal input shows a suggestion popup listing all skills
- [x] Suggestions filter in real-time as the user continues typing (e.g., `/he` matches `/help`)
- [x] Arrow keys navigate suggestions, Enter/Tab selects, Escape dismisses
- [x] Selected skill name is inserted into the input field
- [x] Popup is accessible with proper ARIA attributes

## Technical Constraints
- Next.js App Router + React (canary)
- shadcn/ui + Tailwind CSS for styling
- Must not break existing chat submission flow
- No external autocomplete libraries
