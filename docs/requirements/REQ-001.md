# REQ-001: CLI Bootstrapping

## Description
The ability to run `npx nos` and have it automatically start a local server and open a browser window.

## Acceptance Criteria
- [x] `package.json` configured with a `bin` entry.
- [x] CLI wrapper script (`bin/cli.js`) implemented.
- [x] Automated browser opening on server start.
- [x] Graceful server shutdown on `SIGINT`.

## Technical Constraints
- Must work across macOS, Windows, and Linux.
- Must use the `open` package for cross-platform browser support.
